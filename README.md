📦 S3 Bucket CRUD Operation (Spring Boot + AWS S3)
🚀 Overview

This project is a Spring Boot-based backend application that performs CRUD operations on AWS S3 buckets. It allows users to upload, download, delete, and manage files in an S3 bucket using REST APIs.

🛠️ Tech Stack
Java 17
Spring Boot
AWS S3 (Amazon Web Services)
Maven
REST APIs

⚙️ Features
Upload files to AWS S3 bucket
Download files from S3
Delete files from S3
List objects in bucket
Secure AWS configuration using Spring Boot
